Breast Cancer Prediction using Neural Networks

This project aims to build a predictive model to classify breast cancer tumors as either malignant (M) or benign (B) based on various features derived from cell measurements. The model utilizes a neural network (NN) architecture for binary classification, leveraging a dataset of features that describe the characteristics of cell nuclei in breast cancer biopsies. The final goal is to predict whether a tumor is malignant or benign based on these measurements, helping medical professionals make informed decisions about treatment. The model is built using a fully connected neural network (also known as a Multi-Layer Perceptron, or MLP)

Custom Hand Gesture Detection with YOLO

This project showcases a custom hand gesture detection system built using the YOLO (You Only Look Once) object detection framework. The model is designed to recognize and classify five common hand gestures: FingersCross, OK, PeaceOut, ThumbsDown, and ThumbsUp. This project demonstrates the power of YOLO for real-world applications by training a custom model for specialized object detection tasks. It also emphasizes efficient development practices with Dev Containers, enabling developers to quickly get started with training, fine-tuning, and deploying machine learning models. This setup is ideal for developers, researchers, and enthusiasts looking to explore object detection or create interactive applications that utilize computer vision

The GraphRAG ChatBot

The ChatBot is like a super-smart map that helps answer questions about any document. It breaks the document into small pieces, turns them into nodes, and connects these points with edges if they share ideas. To make this map, I use tools like NetworkX to organize information, a lemmatizer to simplify words, and embeddings to compare ideas as numbers. This chatbot doesn’t just stop at finding related ideas—it goes further. It uses a llm to understand the concepts and connects them using advanced methods like cosine similarity. When you ask it a question, the chatbot explores the map to find the best answers. If the answer isn’t clear, it digs deeper, searching nearby ideas and adding more context until it finds a complete answer. This is like solving a puzzle—piece by piece, it builds the best possible response. Whether it’s for a kid curious about how a machine reads stories or an expert analyzing complex reports, GraphRAG bridges the gap between raw information and clear, thoughtful answers!
